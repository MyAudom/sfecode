បង្កើត​គេហទំព័រ

HTML (Hypertext Markup Language) មានន័យថា ជាភាសាសម្គាល់ webpage លើ internet ដែលត្រូវបានបង្កើតឡើងដោយសមាគមន៍ World Wide Web: W3C។ វាជាវិធីពិសេសមួយដែលយើងប្រើដើម្បីបង្កើតគេហទំព័រ (website) ដូចជា អត្ថបទ រូបភាព និងច្រើនទៀត។ វាដូចជាការប្រើក្រដាសពណ៌ផ្សេងៗគ្នា និងស្ទីកឃ័រដើម្បីរៀបចំសៀវភៅរូបភាពមួយ។ ហើយក៏ជាកូដដំបូងបំផុត ដែលអ្នកដែលចង់ចេះអំពីការសរសេរកូដគេហទំព័រគ្រប់រូបត្រូវសិក្សា ។

ចំណុចសំខាន់ៗរបស់ HTML មាន

• Hypertext: គឺជាតំណភ្ជាប់គេហទំព័រដទៃទៀត

• Markup: គីជាកូដដែលរៀបចំគេហទំព័រអោយមានសសណ្ឋាន

• Language: គឺជាកូដដែលកម្មវិធីអ៊ីនធឺណែតយល់បាន។

ប្រវត្តិខ្លីៗនៃកំណែទម្រង់របស់ HTML

HTML ត្រូវបានបង្កើតឡើងនៅឆ្នាំ 1991 ដោយលោក Tim Berners-Lee។ វាដូចជាការបង្កើតភាសាថ្មីមួយសម្រាប់កុំព្យូទ័រ។ ចាប់តាំងពីពេលនោះមក វាបានកែលម្អជាច្រើនដង ដូចជាការបន្ថែមពាក្យថ្មីៗទៅក្នុងវចនានុក្រម ។

• ដោយ HTML ត្រូវបានកែប្រែទម្រង់ជាច្រើនដង ជាច្រើនសាររួចមកហើយគឺមាន ៖

- HTML 1.0 - (1991)
- HTML 2.0 - (1995)
- HTML 3.2 - (1997)
- HTML 4.01 - (1999)
- HTML 5 - (2012)

Note : HTML ត្រូវបានផ្លាស់ប្តូរកំណែទម្រង់ពីមួយជំនាន់ទៅមួយជំនាន់ឲកាន់តែប្រសើរ និងមានមុខងារថ្មីៗជាច្រើន ។

ធាតុ (Element)

ធាតុ សំរាប់កំណត់សមាសធាតុទៅឲ្យអ្វីមួយ ។ និយាយឲ្យសាមញ្ញ គឺប្រើសំរាប់ប្រើថា អក្សរមួយដុំនឹងជាអ្វី ជាចំណងជើង ជាកថាខណ្ឌ ឬជាអ្វីផ្សេងៗ ។

ឧទាហរណ៍ ៖ បើយើងប្រើធាតុក្រាស់ទៅលើអក្សរ នោះអក្សរនឹងក្រាស់ទៅតាមនឹងដែរ។
