Attribute

Attribute គឺត្រូវបានប្រើដើម្បីផ្ដល់ព័ត៌មានបន្ថែមអំពីធាតុ HTML។ Attributes ជួយកំណត់លក្ខណៈសម្បត្តិ, អាកប្បកិរិយា, ឬលក្ខណៈនៃធាតុ។

ឧទាហរណ៍៖ Attribute អាចប្រើសំរាប់ប្រាប់ថា អត្ថបទក្នុងធាតុនោះ ជាភាសាអ្វី ។

• Style Attribute

Style Attribute គឺត្រូវបានប្រើដើម្បីបន្ថែមរចនាបទទៅអោយធាតុនីមួយៗ ដូចជាការបន្ថែមពណ៌ ការដាក់ពុម្ពអក្សរ ទំហំ និងបន្ថែម CSS អោយទៅធាតុទៅតាមតម្រូវការផ្សេងៗជាច្រើនទៀត។

• Class Attribute

Class Attribute ត្រូវបានប្រើជាញឹកញាប់ដើម្បីចង្អុលទៅឈ្មោះថ្នាក់ក្នុងសន្លឹករចនាប័ទ្ម។ វាក៏អាចត្រូវបានប្រើដើម្បីភ្ជាប់ជាមួយ CSS និង JavaScript ដើម្បីបន្ថែមនូវទំហំ រូបភាព និងផ្សេងៗទៀតដើម្បីរៀបចំធាតុដោយប្រើឈ្មោះថ្នាក់ជាក់លាក់។

• ID Attribute

Id Attribute ត្រូវបានប្រើដើម្បីបញ្ជាក់លេខសម្គាល់តែមួយគត់សម្រាប់ធាតុ HTML ។អ្នកមិនអាចមានធាតុលើសពីមួយដែលមានលេខសម្គាល់ដូចគ្នានៅក្នុងឯកសារ HTML ទេ។ វាក៏អាចត្រូវបានប្រើដើម្បីភ្ជាប់ជាមួយ CSS និង JavaScript ដូចគ្នាទៅនឹង Class ផងដែរ ។
