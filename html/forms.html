Introduction to Forms

HTML Forms គឺជាបច្ចេកទេសសំខាន់មួយក្នុងការអភិវឌ្ឍន៍វេបសាយ ដើម្បីអាចទាក់ទងនឹងអ្នកប្រើប្រាស់ដោយប្រមូល និងផ្ញើទិន្នន័យ។ Form ត្រូវបានប្រើសម្រាប់ការងារដូចជា ចុះឈ្មោះ, ចូលគណនី, ផ្តល់មតិ, និងផ្សេងៗទៀត ។

• នៅក្នុង html forms មានធាតុមួយចំនួនដូចជា៖

- <form> tag
- Action Attribute
- Method Attribute

<form> tag

<Form> Tag គឺជាធាតុដែលប្រើជាប្រអប់សម្រាប់ផ្ទុកឧបករណ៍បញ្ចូលទិន្នន័យដូចជា input, checkbox, radio button, និង button។ វាកំណត់រចនាសម្ព័ន្ធ និងអាកប្បកិរិយារបស់ Form ។

ឧទាហរណ៍៖ អនុវត្ត៖ បង្កើត form មួយសម្រាប់ Submit ។

Action Attribute

action បញ្ជាក់អាសយដ្ឋាន URL ឬ Endpoint ដែលទិន្នន័យនឹងត្រូវផ្ញើចូលទៅបន្ទាប់ពីបញ្ចូល Form។

• ប្រសិនបើ action ជាផ្លូវទាក់ទង សកម្មភាពនឹងត្រូវបញ្ជូនទៅផ្លូវនោះនៅលើដែនដីបច្ចុប្បន្ន។

• ប្រសិនបើទទេ (empty) Form នឹងបញ្ចូនទៅទំព័របច្ចុប្បន្ន។

• ប្រសិនបើជា URL ពេញលេញ ទិន្នន័យនឹងត្រូវបញ្ចូនទៅអាសយដ្ឋានខាងក្រៅ។

ឧទាហរណ៍៖ អនុវត្ត៖ បង្កើត Form ហើយរួចធ្វើរការដាក់ URL ទៅកាន់ទីតាំងណាមួយជាក់លាក់ ។

Method Attribute

method បញ្ជាក់របៀបផ្ញើទិន្នន័យ Form ទៅម៉ាស៊ីនមេ។ មានតម្លៃសំខាន់ 2 គឺ៖

1. GET Method: នឹងបញ្ជូនព័ត៌មានទាំងឡាយទៅកាន់ URL ដែលបានកំណត់ដោយ action attribute ។

• ល្អសម្រាប់សំណើរ (requests) ដែលគ្រាន់តែអាន (read-only) (ឧ. Form ស្វែងរក)

• ទិន្នន័យត្រូវបានបង្ហាញនៅក្នុង URL ដូច្នេះមិនគួរប្រើសម្រាប់ទិន្នន័យសំងាត់។

2. POST Method: ជាវីធីមួយក្នុងការបញ្ជូនព័ត៌មានដែរ តែវានឹងមិនបង្ហាញព័ត៌មានទាំងនោះ ឲ្យឃើញយ៉ាងងាយដូច GET method នោះទេ ។ ជាទូទៅគេប្រើ POST method សំរាប់បញ្ជូនព័ត៌មានសម្ងាត់ដូចជា អក្សរសម្ងាត់ជាដើម ។

• ល្អសម្រាប់ទិន្នន័យសំងាត់ ឬទិន្នន័យមានបរិមាណច្រើន។

• ទិន្នន័យមិនបង្ហាញនៅក្នុង URL។

Note:
1. <input> ជាធាតុដ៏សំខាន់មួយសម្រាប់បង្កើត form ដែលអាចទទួលព័ត៌មានពីអ្នកប្រើប្រាស់។ វាផ្តល់លទ្ធភាពដើម្បីបញ្ចូលទិន្នន័យ និងផ្ញើព័ត៌មានទៅកាន់មជ្ឈដ្ឋាន។
2. <label>
• វាប្រើសម្រាប់កំណត់ label សម្រាប់ input element មួយ។
• Label ជួយពង្រឹងភាពងាយស្រួលក្នុងការប្រើប្រាស់ (accessibility) ដោយធ្វើឲ្យអ្នកប្រើប្រាស់អាចចុចលើ label ដើម្បីផ្ដោតទៅលើ input field ដែលទាក់ទង។
3. for
• Attribute for ផ្សារភ្ជាប់ label ជាមួយ input element ដោយប្រើតម្លៃ id របស់ input។
• ក្នុងករណីនេះ, label ត្រូវបានភ្ជាប់ទៅ input ដែលមាន id="…"។
