Introduction to Forms

HTML Forms គឺជាបច្ចេកទេសសំខាន់មួយក្នុងការអភិវឌ្ឍន៍វេបសាយ ដើម្បីអាចទាក់ទងនឹងអ្នកប្រើប្រាស់ដោយប្រមូល និងផ្ញើទិន្នន័យ។ Form ត្រូវបានប្រើសម្រាប់ការងារដូចជា ចុះឈ្មោះ, ចូលគណនី, ផ្តល់មតិ, និងផ្សេងៗទៀត ។

• នៅក្នុង html forms មានធាតុមួយចំនួនដូចជា៖

- <form> tag
- Action Attribute
- Method Attribute

<form> tag

<Form> Tag គឺជាធាតុដែលប្រើជាប្រអប់សម្រាប់ផ្ទុកឧបករណ៍បញ្ចូលទិន្នន័យដូចជា input, checkbox, radio button, និង button។ វាកំណត់រចនាសម្ព័ន្ធ និងអាកប្បកិរិយារបស់ Form ។

ឧទាហរណ៍៖ អនុវត្ត៖ បង្កើត form មួយសម្រាប់ Submit ។

Action Attribute

action បញ្ជាក់អាសយដ្ឋាន URL ឬ Endpoint ដែលទិន្នន័យនឹងត្រូវផ្ញើចូលទៅបន្ទាប់ពីបញ្ចូល Form។

• ប្រសិនបើ action ជាផ្លូវទាក់ទង សកម្មភាពនឹងត្រូវបញ្ជូនទៅផ្លូវនោះនៅលើដែនដីបច្ចុប្បន្ន។

• ប្រសិនបើទទេ (empty) Form នឹងបញ្ចូនទៅទំព័របច្ចុប្បន្ន។

• ប្រសិនបើជា URL ពេញលេញ ទិន្នន័យនឹងត្រូវបញ្ចូនទៅអាសយដ្ឋានខាងក្រៅ។

ឧទាហរណ៍៖ អនុវត្ត៖ បង្កើត Form ហើយរួចធ្វើរការដាក់ URL ទៅកាន់ទីតាំងណាមួយជាក់លាក់ ។

Lists (បញ្ជី)

Lists គឺជាវិធីដើម្បីដាក់តម្លៃដែលពាក់ព័ន្ធគ្នានៅក្នុងរចនាប័ទ្មដែលធ្វើអោយមានរបៀបរៀបរយ និងមានផាសុខភាពក្នុងការអាន។ វាត្រូវបានប្រើសម្រាប់បង្ហាញធាតុ ឬព័ត៌មានដែលត្រូវបានបញ្ចូលដោយអ្នកសរសេរ ។

• Lists ត្រូវបានបែងចែកជា២គឺ

- Lists មានលំដាប់លំដោយ
- Lists គ្មានលំដាប់លំដោយ

• Lists មានលំដាប់លំដោយ

បញ្ជីមានលំដាប់លំដោយ គឺជាបញ្ជីដែលនៅពីមុខវាត្រូវបានចុះជាលំដាប់លេខ ឬលំដាប់អក្សរ ។ ដើម្បីបង្កើតបញ្ជីនេះ ត្រូវប្រើ <ol> ។ ពាក្យពេញរបស់ ol គឺ ordered list ។

• Lists គ្មានលំដាប់លំដោយ

បញ្ជីគ្មានលំដាប់លំដោយ គឺជាបញ្ជីដែលនៅពីមុខវាត្រូវបានចុះជាត្រេ ឬចំនុច ឬរូបអ្វីផ្សេង ។ ដើម្បីបង្កើតបញ្ជីនេះ ត្រូវប្រើ <ul> ។ ពាក្យពេញរបស់ ul គឺ unordered list ។

Table (តារាង)

Table ក្នុង HTML ត្រូវបានប្រើសម្រាប់រៀបចំ និងបង្ហាញទិន្នន័យដោយមានលក្ខណៈដាច់អាស្រ័យក្នុងទម្រង់ជាជួរដេក និងជួរឈរ។ វាជាឧបករណ៍ដែលធ្វើរការបង្ហាញព័ត៌មានជាច្រើនប្រភេទ ដូចជាសំណុំទិន្នន័យ, ការប្រៀបធៀប, និងប្លង់ផ្សេងៗ ។

+ មុននឹងបង្កើតតារាងបាន យើងត្រូវស្គាល់ពីប្រភេទ Tag ដែលត្រូវបង្កើតតារាងជាមុនសិន៖

• ផ្នែកក្បាលនៃតារាង

ដើម្បីបង្កើតផ្នែកក្បាលនៃតារាង ត្រូវប្រើ <thead> ។ ពាក្យពេញនៃ thead គឹ table header ។

• ផ្នែកកណ្តាលនៃតារាង

ដើម្បីបង្កើតផ្នែកកណ្តាលនៃតារាង ត្រូវប្រើ <tbody> ។ ពាក្យពេញនៃ tbody គឹ table body ។

• ផ្នែកក្រោយនៃតារាង

ដើម្បីបង្កើតផ្នែកក្រោយនៃតារាង ត្រូវប្រើ <tfoot> ។ ពាក្យពេញនៃ tfoot គឹ table footer ។

• ជួរនៃតារាង

ក្នុងការបង្កើតតារាង គឺត្រូវបង្កើតម្ដងមួយជួរដេកៗ ។ ជួរនីមួយៗត្រូវសរសេរនៅក្នុង <tr> ។ ពាក្យពេញនៃ tr គឹ table row ។

• ប្រអប់នៃតារាង

ប្រអប់នីមួយៗនៃតារាងត្រូវសរសេរនៅក្នុង <td> ។ ពាក្យពេញនៃ td គឹ table data ។ សំរាប់ប្រអប់នៃតារាងដែលស្ថិតនៅក្នុងផ្នែកក្បាលនៃតារាង គឺត្រូវប្រើ <th> ។ ពាក្យពេញនៃ th គឹ table head ។
