របៀបកំណត់ធាតុ និងរបៀបបញ្ឈប់ប្រតិកម្មនៃធាតុ

កំណត់ធាតុ

• ធាតុ HTML ត្រូវសរសេរក្នុងសញ្ញា <និង>

• ធាតុក្រាស់គឺ b (“bold”) ដូចនះ ត្រូវសរសេរធាតុ b ក្នុងសញ្ញា <b> ទើបអក្សរដែលបានសរសេរមានស្នាមដិតក្រាស់ដែរ។

បញ្ឈប់ប្រតិកម្មនៃធាតុ

• ដើម្បីបញ្ឈប់ធាតុណាមួយ កុំឲ្យមានប្រតិកម្មតទៅទៀត គឺត្រូវសរសេរធាតុនោះម្ដងទៀត ហើយមានសញ្ញា / នៅពីមុខអក្សរតាងនៃធាតុ ។

• ឧទាហរណ៍៖បើយើងផ្តើមឡើងដោយប្រើប្រាស់នូវ <b> នោះយើងក៏ត្រូវតែបញ្ចប់នូវ </b> ទៅវិញដែរ ។

Note : អក្សរតាងនៃធាតុតែងត្រូវបានហៅជាភាសាអង់គ្លេសថា "tag" ។ ឧទាហរណ៍ ធាតុដែលធ្វើឲ្យអក្សរក្រាស់យើងប្រើអក្សរ b គេតែងហៅថា b tag ។

តួនាទីរបស់ Tag ដែលត្រូវប្រើប្រាស់មួយចំនួន

• HTML Tag

<html> គឺដូចជាឆ្អឹងនៃគេហទំព័រ វាជាសញ្ញាដើមនៃទំព័រប្រភេទ HTML ហើយជាអ្នកស្វែងរកការរចនាសរុបក្នុងឯកសារ HTML និងប្រាប់កុំព្យូទ័រពីរបៀបរៀបចំអ្វីៗនៅលើគេហទំព័រ ។

• Head Tag

<head> មានព័ត៌មានអំពីឯកសារ (ដូចជាចំណងជើង និងមុខអក្សរ) ផ្នែកនេះមានព័ត៌មានពាក់ព័ន្ធដូចជា កូដ (CSS), ស្គ្រីប និងព័ត៌មានផ្សេងៗទៀតដែលមិនបង្ហាញជាសាធារណៈ។

Note : <meta> មិនបង្ហាញជារូបរាងលើគេហទំព័រទេ ដូចនេះត្រូវសរសេរក្នុង ។ ម៉្យាងទៀត Meta tag ប្រើដើម្បីកំណត់ប្រភេទតួអក្សរដែលអាចប្រើបានក្នុងទំព័រដូចជាប្រើ UTF-8 ដែលគាំទ្រអក្សរជាសកល (ដូចជា អក្សរខ្មែរ) ។

• Tittle Tag

<tittle> បង្ហាញចំណងជើងនៅលើផ្ទាំងកម្មវិធីអ៊ិនធឺរណែត ។

• Body Tag

<body> គឺជាមាតិកាសំខាន់ៗដែលត្រូវបានបង្ហាញក្នុងគេហទំព័រដូចជា (អត្ថបទ រូបភាព ឬ តំណភ្ជាប់ជាដើម) កូដមួយចំនួនដែលចម្លើយរបស់វាត្រូវបង្ហាញជារូបរាងលើគេហទំព័រ ត្រូវសរសេរនៅក្នុង body tag ។

• Heading Tag

<h1> និង <h6> ដែលប្រើសម្រាប់កំណត់សរសេរក្បាលរឿងតាមលំដាប់លេខនៅលើគេហទំព័រ។ heading ទាំងនេះផ្ដល់នូវរចនាសម្ព័ន្ធដើម្បីបែងចែកឬរៀបចំឯកសារ ដែល <h1> ជា heading ធំជាងគេហើយ <h6> ជា heading តូចជាងគេ ។

• P Tag

គឺជាកថាខណ្ឌដែលត្រូវសរសេរនៅក្នុងកូដ ។

• A Tag

<a> ប្រើសម្រាប់បង្កើតតំណភ្ជាប់ជាលក្ខណៈ URL។ កាលណាអត្ថបទ រូបភាព ឬវីដេអូរមានតំណភ្ជាប់ នៅពេលដែលចុចលើរបស់ទាំងអស់នោះ វានឹងនាំទៅកាន់ទំព័រដែលបានកំណត់ ។

• Img Tag

<img> នៅក្នុង HTML ត្រូវបានប្រើដើម្បីបញ្ចូលរូបភាពទៅក្នុងគេហទំព័រ។ វាជា tag ដែលមិនត្រូវការ tag បិទ (self-closing tag) ហើយត្រូវបានប្រើយ៉ាងទូលំទូលាយសម្រាប់បន្ថែមមាតិកាទស្សនីយភាពដូចជា រូបថត ក្រាហ្វិក និងរូបតំណាង ។
