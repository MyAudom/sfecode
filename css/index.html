និយមន័យCSS

CSS (Cascading Style Sheets) គឺជាបច្ចេកវិទ្យាដែលប្រើសម្រាប់កំណត់ និងតុបតែងរចនាបទនៃគេហទំព័រ។ CSS អនុញ្ញាតឲ្យអ្នកអាចកំណត់រាង (layout), ពណ៌, ទំហំ, កំណត់អក្សរ (fonts), និងការតុបតែងផ្សេងៗសម្រាប់ធាតុ HTML នៅលើគេហទំព័រ ។

• អ្នកសិក្សា

មេរៀននេះត្រូវបានរៀបរៀងឡើងសំរាប់អ្នកសិក្សា ដែលមានបំណងចង់រចនា ក៏ដូចជាផ្ដល់ភាពរស់រវើកទៅឲ្យគេហទំព័រ ។ ដោយមានភ្ជាប់ជាមួយនឹងវីដេអូមួយចំនួន អ្នកសិក្សានឹងមានភាពងាយស្រួលក្នុងការចាប់យកផ្នែកនានានៃមេរៀន ។ CSS អនុញ្ញាតឲ្យអ្នកកំណត់រចនាបទនៃគេហទំព័រ ដូចជា ពណ៌, ទំហំអក្សរ, ទិសដៅ, និងកំណត់ទីតាំងធាតុ HTML ដែលជួយធ្វើឲ្យគេហទំព័រមានភាពទាក់ទាញ និងប្រកបដោយវិជ្ជាជីវៈ ។

• តម្រូវការ

អ្នកសិក្សាមេរៀននេះ គឺតម្រូវឲ្យចេះកូដ HTML ជាមុនសិន ។ បើអ្នកមិនទាន់ចេះកូដ HTML ទេ អ្នកអាចសិក្សាមេរៀន HTML បន្ថែមនៅលើ Website មួយចំនួនដូចជា W3School, CodeForCambodia,….។

បម្រើបម្រាស់ CSS

CSS គឺប្រើសំរាប់ផ្ដល់ភាពស្រស់ស្អាតទៅឲ្យទំព័រទាំងមូល ។ ដោយប្រើប្រាស់ CSS យើងអាចកំណត់ទំហំអក្សរពណ៌អក្សរ គម្លាតអក្សរ កំណត់ផ្ទៃខាងក្រោយជាពណ៌ ឬជារូបភាព គ្រប់គ្រងប្លង់នៃទំព័រ ហើយថែមទាំងអាចផ្លាស់ប្ដូររូបរាងទំព័រឲ្យប្រែប្រួលទៅតាមទំហំផ្សេងៗនៃឧបករណ៍ដែលអ្នកប្រើប្រាស់កំពុងប្រើ ។

កំណែទម្រង់ CSS

HTML ត្រូវបានកែទម្រង់ជាច្រើនលើកច្រើនសាររួចមកហើយ

កំណែទម្រង់ ឆ្នាំ ការអភិវឌ្ឍន៍
CSS 1 1996 -គឺជាកំណែដំបូងបំផុតរបស់ CSS
-បានណែនាំលក្ខណៈសម្បត្តិបឋមសម្រាប់រចនាទម្រង់ដូចជាអក្សរ, ពណ៌, និងការបែងចែងចន្លោះ ។
CSS 2 1998 -បន្ថែមការគាំទ្រសម្រាប់ប្រភេទមេឌា (ដូចជា screen និង print)
-គាំទ្រព្រមាន pseudo-elements និង pseudo-classes ។
CSS 3 1999 & 2011 -លក្ខណៈសម្បត្តិថ្មីដូចជា transitions, animations, និង transformations។
Media Queries សម្រាប់ការរចនាការឆ្លើយតប (Responsive Design) ។
CSS 4 កំពុងអភិវឌ្ឍន៍ -CSS4 មិនមែនជាកំណែផ្លូវការដូច CSS1, 2, ឬ 3 ទេ។ វាពិតជាច្រើនបន្ថែមលក្ខណៈសម្បត្តិថ្មីទៅលើម៉ូឌុល CSS ដែលមានស្រាប់។
- Selectors ដែល្អប្រសើរឡើង (ដូចជា: has(), :is(), និង :where())។
