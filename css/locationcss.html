ទីតាំងសរសេរ CSS

មាន៣កន្លែងដែលអាចសរសេរ CSS បាន៖

• នៅក្នុងធាតុ

• នៅក្នុង <style>

• នៅក្នុង <style>

នៅក្នុងធាតុ

យើងអាចសរសេរ property នៃ CSS នៅក្នុង style attribute នៅធាតុណាក៏បាន ដែលអ្នកចង់រចនា ។

នៅក្នុង <style>

យើងអាចសរសេរកូដ CSS នៅក្នុង <style> ដែល <style> ត្រូវសរសេរនៅក្នុង <head> របស់ HTML ។

នៅក្នុងឯកសារដាច់ដោយឡែក

យើងអាចសរសេរ CSS នៅក្នុងឯកសារដាច់ដោយឡែកពីឯកសារ HTML ។ ឯកសារដែលផ្ទុកកូដ CSS ត្រូវរក្សារទុកដោយបញ្ចប់ដោយ css ។ ដើម្បីភ្ជាប់ឯកសារ HTML ជាមួយឯកសារ CSS អ្នកត្រូវប្រើ <link> នៅក្នុងផ្នែក <head> នៃឯកសារ HTML។ វាជួយរក្សានៃការរៀបចំកូដដោយបំបែក HTML និង CSS ចេញពីគ្នា។

ភ្ជាប់ឯកសារ CSS

• ប្រើ <link> នៅក្នុង <head>

• rel attribute ប្រាប់អំពីទំនាក់ទំនងរវាងឯកសារដែលភ្ជាប់មកជាមួយនឹងឯកសារបច្ចុប្បន្ន ។

• type attribute ប្រាប់អំពីប្រភេទនៃឯកសារដែលភ្ជាប់មក ។

• href attribute ប្រាប់អំពីទីតាំងនៃឯកសារដែលភ្ជាប់មក ។

អាទិភាពនៃបន្ទះ CSS

យើងអាចសរសេរ CSS នៅ ៣កន្លែង ។ ចុះបើទាំង៣កន្លែងនឹង ត្រូវបានរចនាលើអ្វីមួយជាន់គ្នា តើមួយណានឹងត្រូវបានយកជាការណ៍ ? ក្នុងករណីបែបនោះ វាអាស្រ័យលើអាទិភាពនៃទីតាំងទាំង៣ដែលយើងសរសេរ ។ អាទិភាពរបស់វាគឺដូចទៅនឹងពាក្យថា "ធ្លាក់ដូចទឹកធ្លាក់" គឺពីលើចុះក្រោម ។ មួយណានៅក្រោមគេ គឺមានអាទិភាពជាងគេ ។

១. នៅក្នុងធាតុ ព្រោះសរសេរនៅក្នុងធាតុ តែងតែនៅក្រោមជានិច្ច

២. នៅក្នុង <style> ឬនៅក្នុងឯការដាច់ដោយោឡែក អាស្រ័យលើកន្លែងរបស់វា

• បើ <link> នៃ CSS នៅក្រោម <style> ។ វានឹងដំណើរការកូដដែលនៅក្នុង <link> ។

បើ <style> នៅក្រោម <link> នៃ CSS វានឹងដំណើរការកូដដែលនៅក្នុង <style> ។

ចំណាំ ៖ កូដដែលមិនជាន់គ្នា គឺវាដំណើរការជាធម្មតា ។

អំពី Selector

មុននឹងធ្វើការរចនាលើអ្វីមួយ ដំបូងត្រូវចាប់យកធាតុនោះមក ។ ការចាប់យកអ្វីមួយមករចនា វាត្រូវបានហៅថា Selector ។

ចាប់តាម attribute

ដើម្បីចាប់យកធាតុទាំងឡាយទៅតាម attribute ណាមួយ ត្រូវប្រើសញ្ញា "[]" ហើយត្រូវសរសេរលក្ខណៈនៃ attribute នោះ នៅក្នុង ។

attribute តាង attr

[attr] [title] ចាប់ធាតុដែលមាន title attribute
[attr = តម្លៃ] [target=_blank] ចាប់ធាតុដែលមាន target attribute តម្លៃ _blank
[attr ~= តម្លៃ] [title~=Khode] ចាប់ធាតុដែលមាន tittle attribute តម្លៃមានពាក្យ Khode
[attr |= តម្លៃ] [lang | =Km] ចាប់ធាតុដែលមាន lang attribute តម្លៃផ្ដើមដោយ km
ធាតុដែលនៅក្នុង ធាតុ ធាតុ ចាប់ <p> ដែលនៅក្នុង <main> ត្រូវសរសេរ main p
[attr ^= តម្លៃ] [title^="Welcome to"] ចាប់ធាតុដែលមាន title attribute តម្លៃនាំមុខដោយឃ្លា Welcome to
[attr $= តម្លៃ] img[src$=".jpg"] ចាប់ ដែលមាន src attribute តម្លៃបញ្ចប់ដោយ .jpg
[attr *= តម្លៃ] a[href*=khode] ចាប់ ដែលមាន href attribute តម្លៃមានអក្សរ khode

Note ៖ តម្លៃ attribute ណាដែលមានសញ្ញាដកឃ្លា ត្រូវសរសេរតម្លៃនោះនៅក្នុងអព្ភន្តរសញ្ញា ។
